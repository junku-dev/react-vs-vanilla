# React vs Vanilla
## Try it yourself

### Python server
1. install python
2. create venv
3. `pip3 install flask`
4. run `python3 main.py`
5. run `util/load.py`

### start up react
1. cd into `react_frontend`
2. run `npm install`
3. run `npm run start`

### start up vanilla version
1. open up `vanilla_frontend` in vscode.
2. just use the vscode live server.`
